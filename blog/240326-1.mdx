[Крутая статья](https://lightrun.com/answers/gatsbyjs-gatsby-gatsby-remark-audio-inside-of-gatsby-plugin-mdx-returning-undefined)
И ссылки на файлы пришлось доставать с помощью publicURL:
```
query MyQuery {
  allFile(filter: {name: {eq: "liza-poni"}}) {
    edges {
      node {
        id
        name
        publicURL
      }
    }
  }
}
```
