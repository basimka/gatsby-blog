Добавил поддержку MP3 в MD файлы\
Как обычно все оказалось не тривиально.
Очень помогла статья:
[Крутая статья](https://lightrun.com/answers/gatsbyjs-gatsby-gatsby-remark-audio-inside-of-gatsby-plugin-mdx-returning-undefined)
И ссылки на файлы пришлось доставать с помощью publicURL:
```
query MyQuery {
  allFile(filter: {name: {eq: "liza-poni"}}) {
    edges {
      node {
        id
        name
        publicURL
      }
    }
  }
}
```
Но после публикации, воспроизведение по прежнему не работает:\
[https://bobkov.tech/oeuvre/poni/liza/](https://bobkov.tech/oeuvre/poni/liza/)

Поигравшись еще чуть-чуть я наткнулся на статью:\
[https://github.com/danielmahon/gatsby-remark-relative-images/issues/40](https://github.com/danielmahon/gatsby-remark-relative-images/issues/40)
Где говорилось что не нужно употреблять префикс static, да и вообще не заморачиваться.\
По этому мой окончательно верный путь стал выглядеть как: \
`audio: /liza-poni.mp3`
